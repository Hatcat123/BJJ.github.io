这几天使用git用的疯了，先记录使用desktop的过程，再捋顺一下过程。
使用desktop

file>options>登录账户，选择本地仓库地址

我在电脑创建一个BIG仓库，然后里面是小仓库。每个仓库是一个项目。

>![image.png](https://upload-images.jianshu.io/upload_images/6967995-6c5f75ced0554919.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)


Current repository  选择仓库

Change 查看仓库的变化

History 历史操作

Current branch 选择分支  可以通过这找到本地的文件

fetch origin  操作源吧

下面提交分支的过程Summary Description Commit  提交的时候填上摘要，描述。


在Branch里面可以创建 删除  重命名分支

Repository 里面push pull

会了这几步的组和就差不多完全操作git了。
